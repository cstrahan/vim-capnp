## About

Vim syntax highlighting (and folding) for [Cap'N Proto][capnp] schema files.

## Install

#### Pathogen

```
git clone https://github.com/cstrahan/vim-capnp.git ~/.vim/bundle/vim-capnp
```

#### Vundle

Add to ~/.vimrc:

```
Plugin 'cstrahan/vim-capnp'
```

#### NeoBundle

Add to ~/.vimrc:

```
NeoBundle 'cstrahan/vim-capnp'
```
